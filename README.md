# ペット体調ノート

スマホだけで使う、個人向けのペット体調管理Webアプリです。

このアプリは診断アプリではありません。病名診断、治療判断、薬の指示、「病院に行かなくてよい」という判断は行いません。日々の記録を整理して、動物病院で見せやすくするためのものです。

## 起動方法

### PCで確認する

`pet-health-app` フォルダで、次を実行します。

```powershell
python -m http.server 8080
```

ブラウザで次を開きます。

```text
http://127.0.0.1:8080/index.html
```

### スマホで確認する

スマホ確認では、`index.html` の直接開きではなく、簡易サーバー経由で開いてください。

1. PCとスマホを同じWi-Fiにつなぎます。
2. `pet-health-app` フォルダで、次を実行します。

```powershell
python -m http.server 8080 --bind 0.0.0.0
```

3. PCのIPアドレスを確認します。

```powershell
ipconfig
```

4. `IPv4 アドレス` の数字を使って、スマホのブラウザで開きます。

例:

```text
http://192.168.128.118:8080/index.html
```

PC側のPowerShellは、使っている間は閉じないでください。

Windowsの確認を簡単にするため、`start-phone-server.bat` も用意しています。ダブルクリックすると、スマホで開くURL候補が表示されます。

## 使い方

- ホーム画面から「今日の記録をする」を押して入力します。
- 保存できると「記録しました ✓」と表示され、記録一覧に移動します。
- 「記録を見る」から過去の記録を編集・削除できます。
- 「体重グラフを見る」では、体重を2回以上記録すると折れ線グラフが出ます。
- 「病院で見せる」には、最近の記録や体重、お世話日がまとまります。
- 「設定」で、使う項目だけを表示できます。
- 「バックアップを保存する」で記録をファイルに保存できます。
- 「バックアップから復元する」で保存したファイルを読み込めます。

## 修正しやすいファイル

- `app.js`: 画面表示、記録項目、保存処理があります。
- `styles.css`: 文字サイズ、色、余白、ボタンの見た目があります。
- `index.html`: アプリの土台です。通常はあまり触りません。

項目を増やしたい場合は、主に `app.js` の `FIELD_GROUPS` と `DEFAULT_SETTINGS` を修正します。

設定画面に出す項目名は `SETTING_LABELS` にあります。

## データ保存の仕組み

記録はブラウザ内に保存されます。ログインは不要です。

ただし、ブラウザのデータを削除したり、別の端末で開いたりすると記録は共有されません。PCで記録した内容はスマホには自動では移りません。

大事な記録は、設定画面の「バックアップを保存する」で定期的に保管してください。

## 注意点

- このアプリは診断アプリではありません。
- 病名診断、治療判断、薬の指示は行いません。
- 「病院に行かなくてよい」という判断にも使わないでください。
- 気になる変化がある場合は、動物病院に相談してください。
